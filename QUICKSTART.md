### Common Questions

**Q: How do I change the colors?**
A: Search for `bg-[#F5F5F5]` in index.html and replace with your color.

**Q: How do I add more pages?**
A: Create new HTML files (about.html, services.html) and link from navigation.

**Q: How do I update content?**
A: Edit index.html, save, then git add/commit/push to update live site.

**Q: Can I use a custom domain?**
A: Yes! Follow the "Custom Domain Setup" section in DEPLOYMENT.md

**Q: Where are the images stored?**
A: Currently embedded as base64 in the HTML. For better performance with many images, use external files.
